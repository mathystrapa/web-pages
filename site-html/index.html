Front-end: A programação front-end, também chamada de client-side (lado do cliente), é aquela usada pelo navegador para mostrar um programa ao cliente utilizando linguagens como html, css, javascript. Portanto, a programação front-end é responsável por mostrar um programa ao cliente, o que envolve várias coisas além do design.
Back-end: Já a programação back-end, chamada de server-side (lado do servidor), é aquela usada para fazer os servidores e sistemas funcionarem. É a resposável pela funcionalidade de elementos de um projeto que não são percebidos pelo cliente.
