O que é CSS? CSS, que significa "Cascading Style Sheets" (Folhas de Estilo em Cascata), é uma linguagem de estilo utilizada para descrever como os elementos de um documento HTML (ou XML) serão exibidos em uma página da web. Em outras palavras, o CSS permite que você controle a aparência visual de um site, incluindo cores, fontes, tamanhos, posicionamento e muito mais. Como o CSS Funciona? O CSS funciona aplicando regras de estilo aos elementos HTML. Cada regra consiste em um seletor que define qual elemento será estilizado e um conjunto de propriedades e valores que especificam como esse elemento deve parecer.
