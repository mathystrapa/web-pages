Como criar sites e sistemas em Python

Embora muitos achem que Python só serve para back-end, você pode usálo para criar sites e sistemas super dinâmicos. Nessa breve página, darei um resumo de como fazer isso.

Bibliotecas

As melhores e mais famosas bibliotecas utilizadas para a criação de sites em Python são: biblioteca Request, Framework Flask (estou utilizando essa agora) e claro, o Django. Este último é o mais famoso e utilizado por grandes empresas, como a Google, Netflix, entre outras gigantes. Em breve criarei uma página ensinando cada um desses métodos.
